# Images Directory

Please add your logo and hero images to this directory:

## Required Images:

1. **home-care-logo.png** - Your company logo
   - Recommended size: 200x200px
   - Format: PNG with transparent background

2. **hero-1.jpg** through **hero-4.jpg** - Hero section images (optional)
   - These are optional as the hero section has decorative elements
   - Recommended size: 1200x800px
   - Format: JPG or PNG

## Current Status:
The website will work without these images, but adding them will enhance the visual experience.
